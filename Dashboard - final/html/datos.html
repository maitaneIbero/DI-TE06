Distribución de los elementos

El dashboard nos va a permitir conocer información de nuestra base de datos en forma de tablas, gráficos e informes.

El dashboard se compone de una zona izquierda dónde encontramos dos partes diferenciadas

Filtrado de datos

Para escoger el cliente


Escoger información para mostrar

Información que queremos conocer


En la parte inferior encontramos un botón que nos permite generar informes, al pulsarlo nos aparecerá un menú paraescoger el informe que queremos


En la parte central nos aparecerán la información que hayamos escogido así como el menu de informes
