Informes referentes a la base de datos

En la parte inferior contamos con un botón que nos abrirá un menú con aquellos informes que podemos imprimir. Pulsando cualquiera de las opciones se nos abrirá un informe que podremos imprimir

Importante

Todas las opciones que se van a describir a continuación cuentas con un filtro en la zona superior que permitirá al usuario que el informe muestre los datos particulares de la opción que se escoja

Las 4 opciones disponibles son:

Informe Ventas
Conocer las ventas generales de la empresa

Informe Productos
Información básica sobre el cliente seleccionado

Informe Clientes
Datos de facturación por año en forma de gráfico del cliente seleccionado

Gráfico Anual
Datos de las categorías más consumidas en forma de gráfico del cliente seleccionado

Cuando escojamos una de estas opciones se abrirá una ventana nueva con el informe escogido
